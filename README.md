# ⚡ OfficeKit — Công cụ văn phòng miễn phí

> Bộ công cụ tiện ích dành cho dân văn phòng Việt Nam — không cần đăng ký, không quảng cáo, chạy 100% trên trình duyệt.

## 🛠️ 10 Công cụ có sẵn

- ▣ **Tạo mã QR** — URL, WiFi, danh thiếp điện tử, văn bản
- 🔄 **Chuyển đổi ảnh** — PNG ↔ JPG ↔ WebP ↔ BMP ↔ GIF
- 📦 **Nén ảnh** — Giảm dung lượng, tùy chỉnh chất lượng
- 🔑 **Tạo mật khẩu** — Mạnh, tùy chỉnh, kiểm tra độ bảo mật
- 🎨 **Bảng màu** — HEX ↔ RGB ↔ HSL, tạo palette tự động
- 🍅 **Pomodoro Timer** — Timer 25/5/15 phút, đếm session
- 📝 **Công cụ Text** — Đếm từ, chuyển chữ, sắp xếp dòng
- 🧮 **Tính thuế TNCN** — Gross → Net theo luật Việt Nam 2024
- ✍️ **Chữ ký số** — Vẽ tay hoặc gõ chữ, xuất PNG
- 📐 **Đổi đơn vị** — Độ dài, khối lượng, nhiệt độ, tốc độ

## 🚀 Deploy lên Vercel

```bash
# Option 1: Vercel CLI
npm i -g vercel
vercel --prod

# Option 2: Local development
npm install
npm run dev
```

Hoặc kéo thả thư mục vào [vercel.com/new](https://vercel.com/new) — tự động detect Vite và deploy.

## 🔒 Bảo mật & Quyền riêng tư

Mọi thứ xử lý **hoàn toàn trên trình duyệt** của bạn. Không có dữ liệu nào được gửi lên server.
